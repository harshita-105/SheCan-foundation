# 🌸 She Can Foundation – Landing Page

A modern, responsive landing page built for the She Can Foundation NGO.  
The website is designed to highlight the foundation’s mission, programs, and impact in a clean and engaging way.

---

## 🔗 Live Demo
https://she-can-foundation-taupe.vercel.app/

---

## 🛠️ Tech Stack
- HTML5  
- CSS3 (Custom styling with variables + responsive design)  
- JavaScript (Vanilla JS for interactivity)

---

## ✨ Features

- Fully responsive landing page (mobile + desktop)
- Modern UI with smooth animations and transitions
- Interactive navigation menu (hamburger for mobile)
- Scroll-based reveal animations
- Animated impact counters
- Working volunteer/join form UI (frontend only)
- Testimonials section
- Programs showcase section
- Clean NGO-focused storytelling layout

---

## 📂 Sections in the Website

- Hero Section (mission + intro)
- About Us (vision + values)
- Programs (NGO initiatives)
- Impact Statistics
- Volunteer / Join Us Form
- Testimonials / Stories
- Footer with links & contact
